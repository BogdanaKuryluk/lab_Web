Програмування

 
   public class HelloWorld {
        public static void main(String[] args) {
            // Display "Hello, World!" on the console
           System.out.println("Hello, World!");
         }
     }

Одне з моїх хобі - програмування. Я вивчаю Java та розробляю різні програми на цій мові. Для прикладу, ось невеликий код програми, яка виводить на екран фразу "Hello, World!". Це перша програма, яку зазвичай пишуть при вивченні будь-якої мови програмування.

Як бачите, в програмі використовуються класи та методи мови Java. Клас HelloWorld містить метод main, який викликає метод System.out.println для виводу рядка на консоль. Таким чином, програма виконується та виводить фразу "Hello, World!" на екран.
